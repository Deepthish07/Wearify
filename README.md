# Wearify

## 📦 Project Description

Wearify is an online dress shopping platform designed to provide users with a seamless e-commerce experience. Browse, search, and purchase your favorite clothing items with ease.

## ✨ Features

- **Product Browsing** - Explore a wide range of dresses and clothing items
- **Search & Filter** - Find products easily with search and filtering options
- **User-Friendly Interface** - Clean and intuitive UI for better shopping experience
- **Responsive Design** - Works smoothly on desktop, tablet, and mobile devices
- **Shopping Cart** - Add items to cart and manage your selections
- **Order Management** - Track and manage your purchases

## 🛠️ Tech Stack

- **Frontend**: TypeScript, HTML, CSS, JavaScript
- **Architecture**: Modern web application with component-based design

### Language Composition:
- TypeScript: 70.2%
- HTML: 22.1%
- JavaScript: 5%
- CSS: 2.7%

## 🚀 Getting Started

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn package manager

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Deepthish07/Wearify.git
   cd Wearify
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Start the development server**
   ```bash
   npm start
   # or
   yarn start
   ```

4. **Open in browser**
   - Navigate to `http://localhost:3000` (or the port shown in your terminal)

### Build for Production
```bash
npm run build
# or
yarn build
```
